### 🏆 Featured Projects

* **[IntelliAIDrive: Autonomous Driving Agent](https://github.com/ciellamher/IntelliAIDrive)** – An autonomous driving research platform combining real-time computer vision and reinforcement learning to train and interrogate intelligent vehicle agents under dynamic road environments.
* **[Lumo](https://github.com/darknecrocities/Lumo)** – Co-developed a voice-guided AI route-optimization app for visually impaired users. Won **1st Place at the SkyDev Hackathon** (Next.js, Python, Tailwind CSS).
* **[Algorithmic Memory Allocation Simulator](https://github.com/ciellamher/Virtual-Paging-Algorithms-Simulation)** – An OS memory-management simulator built in Streamlit to visualize page replacement algorithms, frame allocations, and fault patterns, layered with Gemini-powered context analysis.
* **[peeView](https://github.com/darknecrocities/peeview)** – Engineered an AI mobile health interpretation interface that leverages computer vision/AI to analyze urine test strips for instant clinical insights (Flutter, React, Tailwind CSS).
